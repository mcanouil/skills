# Shiny Skills

Skills for Shiny app development in both R and Python.

## Overview

This category contains skills that help with building, styling, and deploying Shiny applications. Skills support both Shiny for R (using bslib) and Shiny for Python frameworks.

## Available Skills

- **[brand-yml](../brand-yml/)** - Create and apply brand.yml files for consistent styling across Shiny apps. Supports automatic brand discovery, custom themes, and integration with both R (bslib) and Python (ui.Theme) Shiny applications.
